Dung Abstract Argumentation Frameworks - Help Page

Compute an extension of a Dung abstract argumentation framework, represented using the Trivial Graph Format (TGF), using the selected semantics (grounded, complete, preferred, and stable), and output the extension in the selected format. If there are multiple extensions, all extensions are listed in the textual output but one extension is selected to be displayed in diagrams. If the framework has no extensions, the textual output will be "NO" and the diagrams will show the framework with none of the arguments highlighted". ("In" arguments are shown filled with green color in the diagrams.)

Using the Trivial Graph Format

Representing Dung abstract argumentation frameworks using the Trivial Graph Format is very simple. Take a look at the examples, and perhaps use the tool to compute and visualize some of their extensions, to see how it is done.

Output Formats

The text output format lists the extensions of the argumentation framework in a plain text format.

The GraphmL format is an XML schema for directed graphs. Graphml is supported by several graph editors and visualizations tools. The Graphml produced by Carneades is intended for use with the yEd graph editor.
