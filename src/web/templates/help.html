Carneades {{.Version}} Help Page

Carneades is a tool for evaluating and visualizing structured argument graphs and Dung abstract argumentation frameworks.

The available commands are:

Eval
Evaluate a structured argument graph
Dung
Compute extensions of a Dung abstract argumentation framework.
