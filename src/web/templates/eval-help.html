Structured Argument Graphs - Help Page

Evaluates a structured argument graph and prints or displays it in the selected output format.

Input Formats

The YAML format is the native format of Carneades 4.x. YAML is a schemeless data interchange format. See the examples.

The Argument Interchange Format (AIF) format is serialized using JSON. See the examples.

The AGXML format is the XML schema used by the annotated corpus of argumentative microtexts project.

The Carneades Argument Format (CAF) is the native format of Carneades 3. See the examples.

The Legal Knowledge Interchange Format (LKIF) is an XML schema developed in the European IMPACT project. LKIF is the native format of Carneades 2, a desktop argument mapping tool with a graphical user interface. Also known as the Carneades Editor.

Output Formats

GraphmL is an XML schema for directed graphs. Graphml is supported by several graph editors and visualizations tools. The graphml produced by Carneades is intended for use with the yEd graph editor.

The DOT format is the native format of the open source GraphViz network visualization project.

The YAML format is the native format of Carneades 4.x. YAML is a schemeless data interchange format. When used as an output format, the arguments and statements in the graph are annotated with their calculated weights and labels (in, out, undecided), respectively. See the examples.

The Portable Network Graphics format (PNG) is a standardized raster graphics file format.

The Scalable Vector Graphics format (SVG) is a XML-based standard for vector graphics supported by web browsers.
