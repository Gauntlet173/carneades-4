Evaluate Abstract Argumentation Framework

Compute an extension of a Dung abstract argumentation framework, represented using the Trivial Graph Format (TGF), using the selected semantics, and output the extension in the selected format. If there are multiple extensions, one is selected. If the framework has no extensions, the text output will be "NO" and the diagrams will show the framework with none of the arguments labelled "in". ("In" arguments are shown filled with green color in the diagrams.)

Limitations: Argumentation frameworks with more than 20 arguments are supported by this server only when using grounded semantics. To try Carneades with larger frameworks using one of the other semantics, you can build and install Carneades on your own computer.
